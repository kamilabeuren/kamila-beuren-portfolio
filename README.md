# Portfólio Pessoal

Página de apresentação desenvolvida em React para exibir informações pessoais, foto de perfil e uma breve descrição profissional.

## Tecnologias Utilizadas

- React
- Vite
- JavaScript
- CSS

## Funcionalidades

- Exibição de foto de perfil
- Apresentação pessoal
- Interface simples e responsiva

## Como executar o projeto

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Instale as dependências:

```bash
npm install
```

3. Execute o projeto:

```bash
npm run dev
```

4. Abra no navegador:

```text
http://localhost:5173
```
