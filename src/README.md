# Portfólio Pessoal

Página de apresentação desenvolvida em React com informações pessoais, foto de perfil e breve descrição profissional.

## Tecnologias utilizadas

- React
- JavaScript
- Vite
- CSS

## Funcionalidades

- Exibição de foto de perfil
- Apresentação pessoal
- Layout responsivo

## Como executar o projeto

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Entre na pasta do projeto:

```bash
cd nome-do-projeto
```

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm run dev
```

5. Abra no navegador:

```text
http://localhost:5173
```
